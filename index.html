PROJETS REALISES

Système de détection de la fatigue du conducteur

Ce projet consiste à développer une application web intégrant une webcam pour surveiller en temps réel les signes de somnolence chez un conducteur. En utilisant des bibliothèques comme OpenCV et Keras, le système analyse les mouvements oculaires et les expressions faciales pour détecter si le conducteur est fatigué. En cas de détection, une alarme est déclenchée via l’interface web, alertant le conducteur pour prévenir les accidents de la route.

Plateforme de recommandation de films avec packages R

Ce projet vise à créer une application web permettant de recommander des films aux utilisateurs en fonction de leurs préférences. En utilisant des packages R comme tidyverse, shiny, et recommenderlab, le système analyse les historiques d’avis et de notations des utilisateurs pour générer des suggestions personnalisées. L’interface web est construite avec Shiny, offrant une expérience interactive et dynamique.

Application de détection de fausses nouvelles en temps réel

Ce projet développe une application web capable d’analyser automatiquement les articles de presse pour détecter s’ils sont authentiques ou falsifiés. En utilisant un modèle d’apprentissage automatique basé sur TF-IDF et un classificateur comme PassiveAggressiveClassifier, l’application traite le texte des articles et fournit une évaluation d’authenticité. L’interface web permet aux utilisateurs de coller un lien ou un texte pour une analyse instantanée.

Détection de fraudes aux cartes de crédit via Python

Ce projet utilise des algorithmes d’apprentissage automatique comme les arbres de décision, les réseaux de neurones artificiels (ANN) et le clustering pour identifier des transactions suspectes. En analysant les comportements de dépenses des utilisateurs à partir d’un ensemble de données réelles, le modèle apprend à distinguer les transactions normales des frauduleuses, permettant ainsi une détection précoce et une alerte automatique.
